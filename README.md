# ESP Playground SDK

A collection of ESP8266 projects for the ESP RTOS SDK.

Runs on a Linux host machine.

***
##### Build toolchain (perform this first):

```c
    source ./configure_esp.sh
```


##### Configure ESP SDK:

```c
    make menuconfig
```

##### Build app:

```c
    make app
```

##### Flash app:

```c
    make flash
```
